# 📍 How to Start
### 1. Clone Repository
```bash
$ git clone 
```
### 2. Backend 환경변수 및 my_settings.py 설정

#### rumz/.env 
```bash
# MySQL
MYSQL_DATABASE=
MYSQL_USER=
MYSQL_PASSWORD=
MYSQL_HOST=
MYSQL_PORT=

# GPT API
GPT_API_KEY=

# TTS
ELEVENLABS_API_KEY=

# AWS S3
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_REGION=
AWS_STORAGE_BUCKET_NAME=
DEFAULT_FILE_STORAGE=
AWS_CLOUDFRONT_DOMAIN=
```

#### rumz/settings.py
```python3
from pathlib import Path
import os

from dotenv import load_dotenv
from datetime import timedelta
import pymysql

pymysql.install_as_MySQLdb()

BASE_DIR = Path(__file__).resolve().parent.parent

load_dotenv()

SECRET_KEY = 

DATABASES = {
    "default": {
        "ENGINE": ,
        "NAME": ,
        "USER": ,
        "PASSWORD": ,
        "HOST": ,
        "PORT": ,
    }
}
```

### 3. docker-compose 실행

#### rumz/
```bash
$ docker-compose
```
